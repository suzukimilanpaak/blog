引数の*がどういう役割をするのか時々分からなくなってしまうのでまとめてみた。

- メソッド内の引数に付ける'*' => 複数の引数をまとめる
- メソッドに渡す引数に付ける'*' => 配列を展開して複数の引数として渡す

```ruby
def caller(*args)
  squash(args)
end
def squash(args)
  puts args.inspect
end
 
caller(:index, :show)
# [:index, :show]
caller(:show)
# [:show]
caller(*[:show])
# [:show]
```
